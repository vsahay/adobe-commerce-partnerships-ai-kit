# Adobe Commerce Partnerships AI KIT

The Adobe Commerce Partnerships AI KIT helps both technical and non-technical audiences understand what a VIP Marketplace feature could look like on a partner's marketplace platform.

The Kit provides:

1. **Visual representations** of VIP Marketplace features.
2. **Tools** that let technical users direct an LLM of their choice to visualize a feature and generate the integration code for it.
